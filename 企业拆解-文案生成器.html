文案模版生成器

🛠 使用说明与可定制性

  • 本网页支持粘贴每一行文章数据，字段间可使用 TAB 或多个空格分隔。
  • 输出模版位于 <script> 中的 result = ... 部分，可自行修改为你常用的格式。
  • 如需更换字段名称（如“Notion”换为“Airtable”），只需修改对应字段标题文字。
  • 若希望支持批量输入、模版切换或导出，可拓展此页面逻辑。

📦 如何修改本网页（导出为 .html 后）：

  1. 右键另存为 文案生成器.html，使用浏览器打开即可使用。
  2. 用文本编辑器打开该文件，搜索 result = `...`。
  3. 情况1（推荐）： 保留模板字符串写法（${title} 等），修改排版和文字即可，例如：
    const result = `《${title}》是关于${topic}的文章，由${author}撰写，来自栏目：${section}`
  4. 变量写法不能删掉 ${...}，但可以调换位置、加前后文字，实现自定义模版。
  5. 若要改为固定模版（非动态），可将整段 result 内容换成固定文本字符串。
